INTRODUCTION

These speech files are sampled at 8 KHz, 16 bit mono.

The source code for sigtowav and wavtosig is available in the Examples/FileIO
folder.

The .wav files were recorded using Audacity : http://audacity.sourceforge.net/.


EXAMPLE

Convert the "QuickBrownFox.wav" speech file to .sig format using wavtosig, as follows :

wavtosig quickb~1

Display the .sig file using sgram from the examples folder as follows :

sgram quickb~1.sig 2 512 74 0.000015 0
